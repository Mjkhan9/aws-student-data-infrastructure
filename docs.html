Project Documentation Overview

Full documentation for the AWS Student Data Infrastructure project, including architecture summary, automation design, execution outputs, and academic review.

Architecture Summary

  • Three-tier VPC architecture with public, private app, and private database subnets
  • NAT gateway for outbound-only internet access
  • RDS PostgreSQL with encryption and private networking
  • Security groups configured with least privilege
  • CloudTrail enabled for all regions

Core Design Goals

  • Maintain full FERPA-aligned data isolation
  • Eliminate all public attack surfaces
  • Enforce IAM least privilege and MFA
  • Ensure full auditability via CloudTrail & VPC Flow Logs
