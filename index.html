Secure AWS VPC & IAM Automation Project

Author: Mohammad Khan • Date: November 14, 2025

Faculty Project Summary

This project delivers a production-style AWS network architecture designed to securely support protected student information systems. It implements a segmented multi-tier VPC, encrypted data flows, identity-based least-privilege controls, and automated IAM provisioning using Python and Boto3.

Evaluated by a faculty review panel, the system demonstrated strong security isolation, efficient access governance, and measurable operational improvements.
